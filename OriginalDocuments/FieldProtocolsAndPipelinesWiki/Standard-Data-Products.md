# Standard Data Products

<p>
This page provides an overview of the <strong>standard data products</strong> generated by APPN UAV processing pipelines, including typical spatial resolution, file formats, and software compatibility. By default, products are delivered in <strong>open or widely supported formats</strong> that can be accessed using <strong>freely available tools</strong>.
</p>

<hr>

<details>
  <summary><strong> ☝️ Guiding principles</strong></summary>

- Products are generated using standardised, documented pipelines to ensure consistency across platforms and sites.
- Output formats prioritise accessibility using freely available and open‑source software.
- Spatial resolution and product definitions are explicit and traceable.
- Products are intended to be suitable for downstream analysis, comparison, and reuse.

</details>
